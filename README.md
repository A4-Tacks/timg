The image viewer on the terminal is based on the VT100 standard.

# Function
- Zoom Picture
- Rotate Picture
- Mirror image
- Inverted image
- Grayscale image
- Change interpolation algorithm
- Change background color
- Adjusting the output color difference threshold to improve output speed
